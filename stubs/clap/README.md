# spicycrab-clap

Python type stubs for the [clap](https://crates.io/crates/clap) Rust crate.

This crate re-exports from: clap_builder

**Install with cookcrab, NOT pip:**

```bash
cookcrab install clap
```

## Usage

```python
from spicycrab_clap import Command, Arg, ...
```

## Dependencies

This package depends on:
- spicycrab-clap_builder
